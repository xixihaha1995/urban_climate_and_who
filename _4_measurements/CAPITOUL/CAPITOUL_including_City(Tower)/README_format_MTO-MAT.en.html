CAPITOUL 2004-2005

1 minute averaged meteorological data from central site : Monoprix tower



Site : 

Toulouse-Alsace - acronym = Ta

Files :

Naming conventions: 

TaAAAAMM_MAT_%60.asc for the data file

TaAAAAMM_MAT_%60.txt for the description file

where :

  • AAAA = year
  • MM = month
  • %60 for 60 seconds

Example : Ta200407_MAT_%60.asc and Ta200407_MAT_%60.txt


Data file contents :

month period


sampling rate : minute


17 columns, separator: tabulation

  1. date, format JJ/MM/AAAA
  2. time, format HHMMSS.SSS
  3. air pressure: hPa; sensor: Vaisala PTB200.
  4. air temperature: Celsius; sensor MP233 VAISALA.
  5. relative humidity: %; sensor HMP233 VAISALA.
  6. Incoming short wave radiation: watts/m2; CM3 sensor of Kipp & Zonen CNR1.
  7. Outgoing short wave radiation: watts/m2; CM3 sensor of Kipp & Zonen CNR1.
  8. Incoming long wave-Red radiation: watts/m2; CG1 sensor of Kipp & Zonen CNR1.
  9. Outgoing long wave radiation: watts/m2; CG1 sensor of Kipp & Zonen CNR1.
  10. rain rate: mm/h; sensor: Optical Rain Gauge
  11. rate rate: mm/h; sensor: tipping bucket rain gauge
  12. wind direction: degree; sensor: GILL sonic anemoter, top of tower
  13. wind speed: m/s; sensor: GILL sonic anemoter, top of tower
  14. wind direction: degree; sensor: GILL sonic anemoter, intermediate level
  15. wind speed: m/s; sensor: GILL sonic anemoter, intermediate level
  16. CO2 concentration: g/m3; sensor: LICOR 7500
  17. Tower position 
    1. = full extension                (height above ground: 48.05 m)
    2. = intermediate extension  (height above ground: 38.98 m)
    3. = intermediate extension  (height above ground: 38.23 m)
    4. = down                             (height above ground: 26.93 m)

Code for missing data : +9999


Frequency : The original data are sampled at 0.1 Hz for pressure, temperature,

              humidity, radiation, rain rate; at 50 Hz for the GILL sonic

              anemometers and at 20 Hz for the LICOR 7500. Then data are

              averaged over one minute period.
